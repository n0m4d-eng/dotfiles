# Readme

- Dotfiles go in this repo. It will be updated as things change, and there might be additions and deletions.
- They are ideally meant to be copy and paste so they're easy to deploy on a new system. But some might require more explaining.

# Nvim custom plugins
- Stopped using a custom lua config for nvim, and went with `nvchad` for a starter config.
- Added 2 new custom plugins for a tag database and pasting images into markdown notes.
    - paste-image
    - tag_picker
- They can be directly copied into the nvchad config for custom plugins.
